Sélecteurs CSS et spécificité

Les différents types de sélecteurs

Nom du sélecteur Ce qu'il sélectionne Exemple
Sélecteur d'élément (parfois appelé « sélecteur de balise » ou « sélecteur de type ») Tous les éléments HTML d'un type donné. p sélectionne tous les

Sélecteur d'ID L'élément d'une page qui possède l'ID fourni (pour une page HTML donné, on ne peut avoir qu'un seul élément pour un ID donné). #my-id sélectionne p id="my-id" ou a id="my-id"
Sélecteur de classe Les éléments d'une page qui sont de la classe donnée (pour une page donnée, il est possible d'avoir plusieurs éléments qui partagent une même classe). .my-class sélectionne p class="my-class" et a class="my-class!"
Sélecteur d'attribut Les éléments de la page qui possèdent l'attribut donné. img[src] sélectionne img src="monimage.png" mais pas
Sélecteur de pseudo-classe Les éléments donnés mais uniquement dans un certain état (par exemple quand on passe la souris dessus). a:hover sélectionne mais uniquement quand le pointeur de la souris est au-dessus du lien.
